# HNG Frontend Stage 0 - Testable Profile Card

This project is a solution to the Stage 0 task for the HNG Internship. It's a simple, responsive, and accessible profile card built with HTML, CSS, and vanilla JavaScript.

## Live URL

You can view the live project here: https://imaginative-marzipan-97aae4.netlify.app/

## Features

-   **Semantic HTML:** Structured with tags like `<article>`, `<figure>`, and `<section>`.
-   **Responsive Design:** Looks great on mobile, tablet, and desktop devices using CSS Grid and Flexbox.
-   **Dynamic Content:** The current UTC time is displayed and updated in real-time using JavaScript.
-   **Testable:** All required elements include `data-testid` attributes for automated testing.

## How to Run Locally

1.  Clone the repository: `git clone https://github.com/gbollybambam/hng-profile-card-task1-frontend.git`
2.  Navigate to the project directory: `cd hng-profile-card-task1-frontend`
3.  Open the `index.html` file in your browser.
